# 03. 뷰

뷰는 `blade` 템플릿을 사용한다.

```php
/**
 * resources/contact/index.blade.php
 */
Contact Us


/**
 * routes/web.php
 */
<?php

Route::get('customers', function () {
    // 뷰파일의 위치와 뷰를 지정하는 방법을 잘 살펴보자
    return view('customers.index');
});

// 위와 완벽하게 같은 방법
Route::view('customers', 'customers.index');
```
